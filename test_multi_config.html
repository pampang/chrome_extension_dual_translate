üìã How to Test Multiple Configurations:

  1. Open the extension popup (click the extension icon)
  2. Select "OpenAI Compatible" from Translation API
  3. Click "+ New Configuration"
  4. Create your first configuration (see examples below)
  5. Click "Save"
  6. Create more configurations by repeating steps 3-5
  7. Use the dropdown to switch between configurations
  8. Click "Translate Current Page" to test
  9. Compare translations from different configurations

API URL: https://api.groq.com/openai/v1

üéØ What to Look For:

  • Configuration Persistence: Configurations should be saved and available after closing/reopening popup
  • Selection Memory: The selected configuration should remain selected
  • Translation Quality: Different configurations may produce different translation quality
  • Speed Differences: Some providers (like Groq) are faster than others
  • Error Handling: Invalid configurations should show appropriate error messages
